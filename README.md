# TCC - Visualização de Indicadores Acadêmicos

Projeto de Trabalho de Conclusão de Curso
Tecnologia em Análise e Desenvolvimento de Sistemas - IFRS

## Objetivo
Desenvolver um sistema de visualização de indicadores acadêmicos utilizando Python.

## Estrutura

- dados_brutos/ → dados originais coletados
- dados_tratados/ → dados após limpeza e transformação
- notebooks/ → exploração e análises exploratórias
- dashboard/ → aplicação de visualização
- relatorio/ → documentos do TCC

## Tecnologias
- Python
- Pandas
- Matplotlib / Seaborn / Plotly
